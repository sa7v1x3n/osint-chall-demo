# OSINT Vercel Challenge

Static HTML challenge project.

## Local preview
Open `index.html` in a browser, or run:

```bash
python -m http.server 3000
```

Then open http://localhost:3000

## Deploy with Vercel CLI

```bash
npm i -g vercel
vercel --prod
```

## Files
- `index.html` - challenge page
- `assets/exhibit-a.png` - uploaded Charminar image
- `vercel.json` - optional clean URL config
